Olá! Meu nome é **Paulo Henrique Silva Vieira**.

Sou **Engenheiro de Software Sênior** com mais de **10 anos de experiência** desenvolvendo sistemas para fintechs, plataformas SaaS, aplicações blockchain e produtos digitais.

Minha principal especialidade é o desenvolvimento **backend** utilizando **Node.js**, **TypeScript** e **NestJS**, sempre buscando criar aplicações escaláveis, resilientes e de fácil manutenção.

Ao longo da minha carreira tive a oportunidade de trabalhar com:

- APIs REST e microsserviços
- Arquitetura de software
- Sistemas distribuídos
- Processamento assíncrono e mensageria
- PostgreSQL, MongoDB e Redis
- RabbitMQ
- AWS e Google Cloud Platform
- Docker e Kubernetes
- Observabilidade com OpenTelemetry, Grafana, Loki e Datadog
- Blockchain e exchanges de criptomoedas

## Sobre este blog

Criei este blog para compartilhar conhecimentos adquiridos durante minha carreira e também nos projetos pessoais que desenvolvo para estudar tecnologias e arquiteturas mais avançadas.

Meu objetivo não é publicar dicas rápidas, mas explicar **como resolver problemas reais de engenharia**, mostrando as decisões técnicas, os desafios encontrados e os motivos por trás de cada solução.

Aqui você encontrará artigos sobre:

- Arquitetura de Software
- Clean Architecture
- Domain-Driven Design (DDD)
- NestJS
- TypeScript
- PostgreSQL
- Microsserviços
- Sistemas Distribuídos
- Mensageria
- Engenharia de Software
- Performance e Escalabilidade

Sempre que possível, os artigos serão acompanhados do código-fonte completo para que qualquer pessoa possa estudar a implementação na prática.
