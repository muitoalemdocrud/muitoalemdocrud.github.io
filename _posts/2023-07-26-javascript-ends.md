***Opa pessoal! Tudo bom com vocês?***

Dessa vez trouxe um texto rápido, sobre uma pergunta que vem sendo feita na bolha tech: Será que javascript ainda tá valendo a pena?

*Nesse texto vou considerar que você já saiba o básico sobre javascript, caso não saiba, corre lá no google dá uma lida e volta aqui para continuar a leitura!*

## Minha experiência com javascript

Tenho cerca de 10 anos na área de tecnologia, grande parte desses anos atuando com desenvolvimento de software. Aprendi muito durante essa jornada e até hoje busco mais conhecimento, pois nessa área não tem espaço para corpo mole ou estagnação.

Umas das principais tecnologias que pagam minhas contas é o javascript. Comecei como muitos usando JS com jquery no frontend (que aliás tá mais vivo do que nunca!). Até aquele momento eu particularmente não conhecia nodeJS e muito menos npm, todas as libs de javascript eram usadas no modo bruto, baixa o código minificado do site e metia no front e nada de usar CDN ok?! Vai que o link do CDN cai e seu site que usava aquele código vai junto?

![Javascript Everywhere2023-07-26-javascript-ends](/assets/img/javascript-ends/javascript-every.jpg)

## O mercado saturou

Com o tempo o javascript ganhou muito mais força por causa do NodeJS e hoje em dia dá pra usar javascript no backend, em aplicativo, em desktop, em arduino, em tudo que é lugar tem javascript! (Me parece o Java com seus 3 bilhões de dispositivos!)

O uso de javascript em várias áreas é muito bom, pois é uma linguagem tranquila para trabalhar e estar em vários lugares amplia as possibilidades para novos desenvolvedores, mas isso pode gerar um pouco de excesso de novos programadores javascript, pois gera uma concorrência gigantesca no mercado. Muitos devs, poucas vagas e para resolver isso a régua do mercado sobe, ficando mais difícil para novos programadores entrarem no mercado.

## Minha opinião

Não estou sendo ingrato com javascript mas creio que ela não seja uma boa porta de entrada para novos programadores, não nesse momento que escrevo esse post, onde o mercado está saturado devs dessa linguagem. Vale a pena investir em uma que não esteja tão saturada ou que te traga um diferencial nos olhos dos recrutadores.

Existem muitas linguagens boas e fáceis de aprender, uma delas é o Ruby com um framework, o Rails, o PHP também é ótima e também tem um framework principal, o Laravel que é tão bom quando o Rails. Temos Java com uma infinidade de aplicações e é uma linguagem que está bem madura, enfim... linguagem não falta.

## Conclusão

Em resumo, não siga a manada, vá na contramão, fazer o que todo mundo faz não traz destaque!

E respondendo a pergunta do título: javascript ainda tá valendo a pena... mas não para você que está começando.

Então é isso pessoal! Comente o que achou, qual sua experiência com javascript? Estão aprendendo ou já dominam?
